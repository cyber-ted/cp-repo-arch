# Arch repo and more
 
 This is an alternate to the AUR 

## How to use
To use this add the following in your /etc/pacman.conf

    [cp-repo-arch]

    SigLevel = Never 
    
    Server = https://raw.githubusercontent.com/cyber-ted/$repo/main/$arch

This should be good.
